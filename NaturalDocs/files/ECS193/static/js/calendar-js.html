SaveSchedule

function SaveSchedule(overwrite)

saves the currently displayed schedule to the database with the provided name as the primary key.  If ‘overwrite’ is false and the name already exists the user will be prompted if they wish to overwrite the saved schedule.  If ‘overwrite’ is true, an Ajax call will be made to delete the old schedule then an Ajax call will be made to save the currently displayed schedule.

Parameters

overwriteboolean on whether or not to overwrite an entry of the same name in the database

Returns

none
